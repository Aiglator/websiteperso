Mes projets

Gitdiscover

Gitdiscover est un outil qui automatise la structuration et le versionnement des projets en intégrant Git et l'IA. Il repose sur TinyNLP ainsi qu'une IA spécialement conçue pour collaborer avec TinyNLP et Git local, offrant ainsi une gestion intelligente et optimisée du versionnement.

Apibooks

ApiBooks est un outil léger et intelligent pour documenter automatiquement vos API Express. Basé sur un fichier openapi.json, il génère une interface web moderne pour consulter vos routes — sans créer de fichier HTML statique. L’interface est intégrée à votre serveur via une URL personnalisable comme /docs, et reste responsive et modifiable selon vos besoins.
